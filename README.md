# myHealthGuides v1.1

*A few seconds a day. One insight at a time.*

## Overview
**myHealthGuides** is a React Native Expo application designed to help individuals capture daily health signals, recognize meaningful patterns, and prepare for more productive conversations with their healthcare providers.

The platform emphasizes calm awareness rather than constant monitoring, empowering users to build their personal “Health Owner’s Manual” over time.

## Core Modules

### 🩺 myCareGuide
Capture daily health signals such as sleep, mood, activity, and optional home measurements.

### 💊 myMedGuide
Track medications, log doses, and check for potential interactions with supplements or over-the-counter products.

### 📊 myInsights
Transform repeated signals into meaningful patterns and reflections to support better health understanding.

### 🗂️ myVisitGuide
Generate clear summaries and questions to help users prepare for medical appointments.

## Key Features

- 📱 Built with **React Native** and **Expo**
- 🧩 Modular architecture with separated screens and logic
- 🧠 Insight-driven design focused on pattern recognition
- 🔒 User-controlled data sharing
- ⚡ Fast interactions: “One screen, three taps, five seconds”
